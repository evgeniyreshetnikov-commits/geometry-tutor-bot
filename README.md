# Geometry 7 Class Telegram Bot

Телеграм-бот по геометрии 7 класса.

Что умеет:
- объясняет темы простыми словами;
- не даёт готовые решения;
- помогает найти идею и сделать следующий шаг;
- поддерживает PDF, DOCX, JPG, PNG;
- даёт тренировку, мини-викторину, задание дня;
- показывает прогресс и достижения в рамках текущего запуска бота.

## Запуск в Railway

1. Загрузите файлы в GitHub.
2. Подключите репозиторий в Railway.
3. В Variables добавьте только:
   - `BOT_TOKEN` = токен от BotFather
4. Нажмите Deploy.

## Зависимости

Установить локально:

```bash
pip install -r requirements.txt
```

Для OCR на Railway добавьте переменную:

- `RAILPACK_DEPLOY_APT_PACKAGES` = `tesseract-ocr tesseract-ocr-rus`

## Важно

Прогресс, звёздочки и достижения хранятся только во время текущего запуска бота.
После нового деплоя Railway они могут сброситься.
